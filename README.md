# ⚽ L Clasico

Современный веб-сайт для отслеживания футбольного соревнования между двумя командами класса, выполненный в стиле приложения FotMob.

## 🎯 Возможности

### Для всех пользователей:
- 📊 **Турнирная таблица** - полная статистика команд (матчи, победы, ничьи, поражения, голы, очки)
- ⚽ **Последние матчи** - красивые карточки с результатами всех проведённых матчей
- 📱 **Адаптивный дизайн** - отлично работает на телефонах, планшетах и компьютерах
- 🎨 **Тёмная тема** - современный дизайн в стиле FotMob

### Для администратора:
- 🔐 **Безопасный вход** - доступ к админ-панели по логину и паролю
- ➕ **Добавление матчей** - простая форма для добавления новых результатов
- ✏️ **Редактирование** - возможность исправить результаты матчей
- 🗑️ **Удаление** - удаление ошибочно добавленных матчей
- 🔄 **Автоматический пересчёт** - таблица обновляется мгновенно

## 🚀 Быстрый старт

### Способ 1: Просто открыть файл (рекомендуется)

1. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari)
2. Готово! Сайт работает без установки дополнительного ПО

### Способ 2: Локальный сервер (опционально)

Если вы хотите запустить через локальный сервер:

```bash
# С помощью Python 3
python -m http.server 8000

# С помощью Node.js (если установлен npx)
npx serve

# С помощью PHP
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`


## 📖 Как использовать

### Просмотр таблицы и матчей
1. Откройте `index.html` в браузере
2. Вы увидите турнирную таблицу и список последних матчей
3. Никакой регистрации не требуется!

### Добавление матча
1. Нажмите кнопку **"Админ"** в правом верхнем углу
2. Введите логин и пароль
3. Нажмите **"Добавить Матч"**
4. Заполните форму:
   - Выберите команды
   - Введите счёт
   - Укажите дату матча
5. Нажмите **"Сохранить"**
6. Таблица обновится автоматически!

### Редактирование матча
1. Войдите в админ-панель
2. Найдите нужный матч в списке
3. Нажмите кнопку **"✏️ Изменить"**
4. Внесите изменения
5. Сохраните

### Удаление матча
1. Войдите в админ-панель
2. Найдите матч для удаления
3. Нажмите кнопку **"🗑️ Удалить"**
4. Подтвердите удаление

## 💾 Хранение данных

Все данные (матчи и статистика) сохраняются в **localStorage** браузера:
- ✅ Данные сохраняются автоматически
- ✅ Данные остаются после перезагрузки страницы
- ⚠️ Данные привязаны к конкретному браузеру
- ⚠️ Очистка кэша браузера удалит все данные

### Резервное копирование данных

Чтобы не потерять данные, можно экспортировать их из консоли браузера:

```javascript
// Открыть консоль (F12) и выполнить:
console.log(localStorage.getItem('matches'));
console.log(localStorage.getItem('teams'));
```

Скопируйте результат и сохраните в текстовый файл.

## 🔧 Настройка

### Изменение названий команд

Отредактируйте файл `app.js` (строки 11-32) и `index.html` (строки 90-95):

```javascript
// В app.js:
let teams = [
    {
        name: 'Ваше название 1',  // Измените здесь
        // ...
    },
    {
        name: 'Ваше название 2',  // Измените здесь
        // ...
    }
];
```

### Изменение цветовой схемы

Отредактируйте CSS переменные в файле `styles.css` (строки 7-15):

```css
:root {
  --color-primary: #00d664;    /* Основной цвет (зелёный) */
  --color-warning: #ffd700;    /* Цвет ничьих (жёлтый) */
  --color-danger: #ff4444;     /* Цвет поражений (красный) */
  /* ... */
}
```

## 📝 Примечания

- Первый запуск создаст два примера матчей для демонстрации
- Все расчёты (очки, разница мячей, форма) происходят автоматически
- Система очков: **Победа = 3 очка**, **Ничья = 1 очко**, **Поражение = 0 очков**
- Таблица сортируется по очкам, затем по разнице мячей

## 🆘 Помощь

### Проблема: Данные не сохраняются
- Убедитесь, что в браузере включены cookies и localStorage
- Проверьте, что не используется режим инкогнито

### Проблема: Не могу войти в админку
- Проверьте правильность логина и пароля
- Логин: `admin`, пароль: `admin123` (по умолчанию)

### Проблема: Таблица не обновляется
- Обновите страницу (F5)
- Очистите кэш браузера и перезагрузите страницу
